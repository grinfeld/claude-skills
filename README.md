# Claude Skills

A collection of Claude Code skills for use with the [Superpowers](https://superhuman.ai) plugin system.

## Skills

| Skill | Description |
|-------|-------------|
| `architect` | System design and architecture guidance |
| `big-data-dev` | Big data engineering best practices |
| `devops-senior` | DevOps and infrastructure expertise |
| `java-senior-dev` | Senior Java development patterns and practices |
| `python-senior-dev` | Senior Python development patterns and practices |
| `rust-senior-dev` | Senior Rust development patterns and practices |
| `scala-senior-dev` | Senior Scala development patterns and practices |
| `senior-dev` | General senior developer guidance |

## Usage

Place this repository's contents under `~/.claude/skills/` or configure your Claude Code plugin to load skills from this directory.

Each skill lives in its own subdirectory and contains a `SKILL.md` file that Claude Code loads when the skill is invoked.
